ECMA

empresa que criou e atualmente mantem a linguagem

Constantemente a ECMA lança features novas para o JavaScript

Babel

Transpilador que faz com que a versão gerada do código seja a mais compatível com todos os browsers

ES1 (1997): o lançamento oficial do JavaScript, com sua primeira versão estável, a 1.1

ES2 (1997): edição mais editorial para ficar de acordo com a ISO/IEC-16262. Não houve adições, remoções, mudanças ou omissões nela

ES3 (1999): primeira atualização real da linguagem, onde houve atualizações de Strings, suporte para expressões regulares, try-catch e afins

ES4 (2008): foi planejada uma completa reestruturação da linguagem, com tantas mudanças que a especificação resultante era quase uma nova linguagem, muitas foram descartadas e outras foram divididas entre as duas versões

ES5 (2012): passou a dar vida ao que temos de JavaScript atualmente, nessa versão foram implementados suporte à JSON, strict, getters and setters, etc.

ES6 (2015): a partir desta versão, nem todos os navegadores possuem suporte para ela, para tal, precisa-se de Babel para utilizar ela. Temos aqui algumas features legais como const, let, arrow functions, classes e promisses

ES7 (2016): Atualmente está é a versão sendo trabalhada, por tanto, nem todas funcionalidades ainda foram implementadas
